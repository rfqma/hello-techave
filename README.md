## Getting Started

install dependencies:

```bash
npm install
```

run dev mode:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with browser.

## Highlights

- **Next.js**
- **React**
- **React Context**
- **Material UI**
- **Tailwind CSS**
- **Form Validation**
- **Responsive**

## Contents

- **Home**: Home page with a simple information

- **Form**: Page to input profile data

- **About**: Page to display profile data
